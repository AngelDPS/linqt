Detailed Description

Define the different types of the program.

A class for a regular on-the-fly hamiltonian matrix.

Defines the index object Defines the lattice geometry

This class is used to store the regular part of the Hamiltonian. It bassically store only the energy parameters, and generate the initial and final index states on the fly. The operations are CPU intensive, but RAM friendly. Generalize this class for a general matrix

Note
This class is still in development process, use it at your own risk!
