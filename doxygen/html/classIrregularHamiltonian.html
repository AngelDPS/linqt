Detailed Description

Defines the types of the program.

Defines the SparseMatrix class The Irregular Hamiltonian Class

This class is the responsible of store the irregular part of a Hamiltonian in a sparse matrix format. It also contain the implementation of the Hamiltonian*Vector product which is extensively used in iterative programs and also contain the implementation used to rescale the spectrum between (-alpha,alpha) interval
