The Chebyshev Iteration Function.

This functions compute the iteration of the Chebyshev vectors following the rule:
T_{n}(x)= 2x T_{n-1}(x) - T_{n-2}(x) It takes as parameters:
The lattice vector used to extract the regular and irregular part of the Hamiltonian
The set id of of the set where the iteration will be performed The Chebyshev truncation order
Initialize the truncation order in a default value 100
